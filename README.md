# Projekt-i-data-science

Mitt projekt handlar om Spotifys ekonomi. Jag vill undersöka hur mycket mer Spotify tjänar om de lyckas få sina gratislyssnare att börja betala för Premium. För att göra det har jag byggt ett program i Python som tar deras riktiga siffror och räknar ut hur vinsten förändras. Jag har också lagt till en simulering för att se hur resultatet ser ut om man räknar på tusentals olika scenarier.

Innehåll:
1. Python fil för själva koden.
2. Rapport om projektet.
